# README

Here you can see the procedure to set up the web app locally

## Tools

* Ruby version 2.0.7
* Rails version 5.2.2
* Database used => SQLite3

### delete database

* **rails db:drop**

### create database

* **rails db:create**

### migrate database

* **rails db:migrate**

## Local deployment instructions

* get in the project folder => **cd mtpb/**
* launch the dev server => **rails server**
* go to => **localhost:3000**
